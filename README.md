Attack on Titans (Java Game):

Overview:

Attack on Titans is a fan-made Java game inspired by the anime and manga Attack on Titan. Players control members of the Scout Regiment, battling Titans using maneuver gear and strategic combat mechanics.

The game is implemented entirely in Java, using Java Swing / AWT for graphics and input handling. It provides fast-paced action gameplay reminiscent of the anime’s iconic Titan battles.

Features:

2D/3D-inspired Action Gameplay – Experience dynamic combat with Java graphics.

Playable Characters – Control iconic characters like Eren, Mikasa, and Armin.

Titan Battles – Fight against Titans of different sizes and behaviors.

Omni-Directional Maneuver Gear – Simulate vertical and horizontal movement.

Score & Ranking System – Track your performance across multiple battles.

Simple Java-Based Engine – No external dependencies required.

 Requirements:

Java Development Kit (JDK) 8 or higher

Operating System: Windows, macOS, or Linux

Recommended IDE: Eclipse, IntelliJ IDEA, or NetBeans
